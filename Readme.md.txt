# 🚀 Ejercicios de Python

Este repositorio contiene mis primeros ejercicios de práctica en **Python**.  
La idea es registrar mi progreso y ver cómo voy mejorando con el tiempo.  

## 📂 Estructura

- **Colecciones/** → ejercicios usando listas, diccionarios, etc.  
- **Estructuras de control/** → condicionales y bucles.  
- **MiniProyecto/** → ejercicios un poco más largos para aplicar todo lo aprendido.  

## 📌 Temas cubiertos
✅ Condicionales (`if/elif/else`)  
✅ Bucles (`for`, `while`)  
✅ Funciones  
✅ Manejo de entradas (`input`)  
✅ Operadores lógicos y aritméticos  

## 🎯 Objetivo
Este repo no busca mostrar proyectos grandes, sino un **registro de mi aprendizaje**.  
Con el tiempo iré agregando ejercicios más complejos y mini proyectos.
